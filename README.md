# AndroidWithLinuxServer-Kotlin-  
My Practice Project

개인 공부를 위한 리눅스 서버 기반의 안드로이드 프로그램  
- Apache2, Mysql, PHP, Android Studio 사용  
- 개발언어 Kotlin 사용  

※기본 기능은 같은 이름을가진 Java 프로젝트와 같음  
※개인적으로 Kotlin을 학습하면서 배운 내용을 바탕으로 기존의 Java 프로젝트를 Kotlin 프로젝트로 변환한 프로젝트임
